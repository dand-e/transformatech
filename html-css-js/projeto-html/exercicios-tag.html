HTML

        HTML é uma linguaguem de marcação utilizada para desenvolver sites para a web.
        É composto por elementos definidos por meio de tags e suporta estilização e scripts externos como por exemplo JavaScript.

CSS

O CSS é a sigla para Cascade Style Sheet ou folha de estilo em cascata. É utilizado para estilizar componentes de uma página HTML.

Pode ser utilizado inline dentra da tag, ou fora dela em uma tag style ou até mesmo em um arquivo externo
