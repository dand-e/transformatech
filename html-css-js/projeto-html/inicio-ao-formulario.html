Sobre formulários HTML

Na tag {form} podem ser definidos atributos como method e action:

  • action: define o local (através de uma URL, por PHP para enviar dados para um banco de dados) ao qual serão enviados os dados do formulário;
  • method: define o método HTTP do formulário. sendo os métodos GET (para obter/buscar/recuperar) e POST (para postar/inserir).

Métodos POST E GET

Os métodos são utilizados para fazer requisições para os servidores, por meio de uma URL, contendo um cabeçalho e um corpo.

  • cabeçalho: contém conjuntos de metadadoscom os recursos do navegador;
  • corpo: tem os dados s serem enviados para o servidor.

Elementos

  • input: define um campo de entrada de dados. pode assumir vários tipos como button, email, password, text, number, file, etc;
  • textarea: define uma area de texto, podendo conter diversas linhas de texto;
  • button: define um botão;
  • selec: define uma lista selecionável, também conhecida como drop-down;
  • option: define uma lista de opções dentro de um drop-down;
  • optgroup: define um grupo de opções;
  • fielset: define um grupo de campos;
  • output: define elementos de saída para o formulário;
  • legend: define um título para o conjunyo de campos;
  • Label: define um rótulo ou legenda para um campo ou controle do formulário;
