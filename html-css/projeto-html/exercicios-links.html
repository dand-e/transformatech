Links

Veremos hoje sobre links em HTML.
Links podem ser imagem, texto ou botões, mas qualquer elemento HTML pode ser um link.
Seja um direcionamento interno ou externo.

Sintaxe

A tag a é usada para identicar o link, seguido de href="caminho do link".

Links geralmente aparecerão da seguinte forma:

  • Não visitado: azul e sublinhado
  • Visitado: roxo e sublinhado
  • Ativo: vermelho e sublinhado

Para links internos não é necessário digitar o site inteiro. Pode-de usar {a href="paginas/pagina1.html"}Link para a página 1{/a} ou se estiver no diretorio anterior {a href="../pagina1.html"}Link para a página 1{/a}

Target

Por padrão a página vinculada será exibida na janela atual do navegador, mas considerando user experience isso pode mudar.

Possibilidades:

  • _self: padrão e abre o documento na mesma janela ou guia
  • _blank: abre o link em uma nova janela ou guia
  • _parent: abre o documento no quadro pai
  • _top: abre o documento em todo o corpo da janela

URL absoluta vs. URL relativa

URL absoluta é quando há o link completo, a relativa é usada para acessar diretórios dentro do própria página
